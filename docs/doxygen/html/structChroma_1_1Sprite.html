Detailed Description

Animated or Static Sprite.

Class for Game Sprite. This sprite can be static or animated.

Load the sprite from file.

This function loads the sprite from a file, if it exists. It supports Aseprite(.ase) sprites/animations as well as static .png, and .jpg files.

Returns
Returns true if the sprite was loaded succesfully.
