Detailed Description

Game object Component interface.

In order to implement a new component, Name() MUST be implemented. In order for the component to interface with the editor, DrawImGui() can be overidden.

void Chroma::Component::DrawComponentValue ( std::string  label)
protected

Draws a component lable for the Editor component widgets.

This will draw the label in the left column of the componet widgets, and the next ImGui component will be drawn on the right. Usage:

DrawComponentValue("Name");
ImGui::Text("##name", Name);

Note: ## is added in front of the ImGui::Text label so that the label is not drawn.
