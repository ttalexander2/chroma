Detailed Description

Logging class, using the spdlog interface.

The logging api provides multiple macros for logging at different levels. Levels occur in the following order: Trace, Info, Warning, Error, and Fatal, from least to most significant.

spdlog can be configured in such a way to only provide warnings at a certain level.
