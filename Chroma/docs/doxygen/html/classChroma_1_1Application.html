Detailed Description

Main class for a Game in the Chroma Engine.

This class handles the update loop for the game, as well as executing all backend events. Overriding this class will create the entry point for your game. In order to set the entry point, see the following example:

Chroma::Application* Chroma::CreateApplication()
{
return new Runtime();
}
